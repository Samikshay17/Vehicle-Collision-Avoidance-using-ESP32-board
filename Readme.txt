# 🚗 Vehicle Collision Avoidance System

## About
A hardware project that detects obstacles and avoids collisions
using ultrasonic sensors and Arduino.

## Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- Motor Driver (L298N)
- DC Motors
- Buzzer/LED (optional)

## How It Works
1. Ultrasonic sensor detects obstacles ahead
2. Arduino processes the distance
3. Motors stop or turn to avoid collision
